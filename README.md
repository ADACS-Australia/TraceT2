# TRACE-T

TRACE-T, or Transient RApid-response using Coordinated Event Triggering, is a web application used to trigger time-critical observations of the sky in response to transient events such as gamma ray bursts or gravitational wave detections.

TRACE-T listens for notices from upstream, early detection instruments and, based on user-configurable conditions, can automate the triggering of downstream observatories such as the Murchison Widefield Array.

## Documentation

The full documentation of TRACE-T is available at [https://adacs-australia.github.io/TraceT2](https://adacs-australia.github.io/TraceT2).
